军团要塞2Linux服务端

参考资料: GSM原文 wiki

强烈推荐直接看原文

安装依赖

CentOS 64-bit

yum install tmux mailx postfix curl glibc.i686 libstdc++ libstdc++.i686

CentOS 64-bit

yum install tmux mailx postfix curl libstdc++

其他发行版的依赖请看原文不在这赘述了

安装本体

1.创建新用户

adduser tf2server
passwd tf2server
su - tf2server

2.安装

wget http://gameservermanagers.com/dl/tf2server
chmod +x tf2server
./tf2server install

启动辣

1.开始

./tf2server start

2.停止

./tf2server stop

3.重启

./tf2server restart

4.命令模式

./tf2server console

使用快捷键"CTRL+b d"退出

5.升级

./tf2server update

5.5验证完整性

./tf2server validate

注意哦mapcycle.txt这样的文件会被重置
